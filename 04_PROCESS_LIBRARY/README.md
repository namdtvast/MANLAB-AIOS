# 04 — PROCESS LIBRARY ⭐ (Quy trình — Hub)

**Ý nghĩa:** 38 quy trình MP01–MP38, mỗi MP là Hub điều hướng — không chứa nội dung, chỉ trỏ tới tầng khác.

**Lưu file gì ở đây:**
- Mỗi `MPxx_Slug/` đúng 3 file: README.md, manifest.yaml, links.yaml
- _DIGITAL_THREADS/*.yaml — chuỗi số xuyên suốt dịch vụ

**KHÔNG lưu ở đây:**
- Nội dung thủ tục đầy đủ (.docx) — chỉ link
- Bản copy biểu mẫu/module/bằng chứng — chỉ trỏ đường dẫn

**Lưu ý:** Thấy file thủ tục/biểu mẫu nằm trực tiếp trong MPxx → sai chỗ. Mxx số hóa MPxx cùng số.
