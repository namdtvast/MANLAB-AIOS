# 04 — PROCESS LIBRARY ⭐ (Quy trình — Hub)

**Ý nghĩa:** 38 quy trình MP01–MP38, mỗi MP là một 'Hub' điều hướng — không chứa nội dung, chỉ trỏ tới các tầng khác.

**Lưu file gì ở đây:**
- Mỗi `MPxx/` đúng 3 file: `README.md` (mục đích, chủ sở hữu, điều khoản ISO, năng lực), `manifest.yaml` (siêu dữ liệu máy đọc), `links.yaml` (đường dẫn tới 03/05/06/07/08/11)
- `_DIGITAL_THREADS/*.yaml` — định nghĩa chuỗi số xuyên suốt dịch vụ (vd. `DT-21`)

**KHÔNG lưu ở đây:**
- Nội dung thủ tục đầy đủ (.docx) — không để ở Hub; thủ tục thuộc hệ tài liệu, Hub chỉ **link**
- Bản copy biểu mẫu, module hay bằng chứng — chỉ trỏ đường dẫn, **không copy**

**Lưu ý:** Nếu thấy một file thủ tục/biểu mẫu nằm trực tiếp trong `MPxx/` → đặt sai chỗ. Mxx số hóa MPxx cùng số.
