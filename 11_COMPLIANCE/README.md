# 11 — COMPLIANCE ⭐ (Tuân thủ & Audit)

**Ý nghĩa:** Lớp chứng minh tuân thủ — nơi đánh giá viên (BoA/VILAS, cơ quan QLNN) truy bằng chứng.

| Thư mục con | Lưu gì |
|---|---|
| `01_ISO_Mapping` | Ánh xạ điều khoản ISO ↔ MP ↔ bằng chứng |
| `02_Legal_Mapping` | Ánh xạ pháp luật ↔ hoạt động |
| `03_Evidence` | Bằng chứng tuân thủ (EV-xxx): hồ sơ, nhật ký, ảnh |
| `04_Audit` | Hồ sơ đánh giá nội bộ & bên ngoài |
| `05_NC` | Sự không phù hợp |
| `06_CAPA` | Hành động khắc phục/phòng ngừa |
| `07_Management_Review` | Hồ sơ xem xét lãnh đạo |
| `08_Risk` | Sổ rủi ro |
| `09_Opportunity` | Sổ cơ hội cải tiến |

**KHÔNG lưu trong tầng này:**
- Bản nháp tài liệu hệ thống (→03/04)
- Biểu mẫu trắng (→06)

**Lưu ý:** Mọi bằng chứng nên truy ngược được về MP đã sinh ra nó (qua mã MPxx).
