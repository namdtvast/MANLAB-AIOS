# 11 — COMPLIANCE ⭐ (Tuân thủ & Audit)

**Ý nghĩa:** Lớp chứng minh tuân thủ — nơi đánh giá viên (BoA/VILAS, cơ quan QLNN) truy bằng chứng.

| Thư mục con | Lưu gì |
|---|---|
| `ISO_Mapping` | Ánh xạ điều khoản ISO ↔ MP ↔ bằng chứng (đã có bảng seed). |
| `Legal_Mapping` | Ánh xạ yêu cầu pháp luật ↔ hoạt động của Viện. |
| `Evidence` | Bằng chứng tuân thủ (EV-xxx): hồ sơ, nhật ký, ảnh chụp. |
| `Audit` | Hồ sơ đánh giá nội bộ và bên ngoài. |
| `NC` | Hồ sơ sự không phù hợp (nonconformity). |
| `CAPA` | Hành động khắc phục/phòng ngừa và theo dõi hiệu lực. |
| `Management_Review` | Hồ sơ xem xét của lãnh đạo. |
| `Risk` | Sổ rủi ro và biện pháp kiểm soát. |
| `Opportunity` | Sổ cơ hội cải tiến. |

**Lưu ý:** Mọi mục bằng chứng nên truy ngược được về MP đã sinh ra nó (qua mã MPxx).
