## Quy tắc cập nhật

| Loại giấy tờ | Tần suất | Cách cập nhật |
|---|---|---|
| Quyết định, chứng chỉ | Liên tục (khi có bản mới) | Lưu file mới với năm: `YYYY_TenFile.pdf` |
| GCN đăng ký KD, KHCN | Hằng năm (gia hạn) | Thêm phiên bản mới, lưu tên + "Lần X" |
| Thẻ kiểm định viên | Hằng năm | Thêm lần cấp mới: `QD_The_Lan_N.pdf` |
| Hồ sơ nhân lực | Hằng quý | Lưu theo định dạng: `YYYY_T_HSNL.pdf` |
| Profile/Giới thiệu | Theo nhu cầu (ít thay đổi) | Ghi rõ ngôn ngữ & tư cách (cá nhân/tổ chức) |
