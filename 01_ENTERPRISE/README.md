# 01 — ENTERPRISE (Tầng doanh nghiệp)

**Ý nghĩa:** Bản sắc và định hướng của Viện ETV — trả lời 'vì sao Viện tồn tại và đi về đâu'.

**Lưu file gì ở đây:**
- Các file định hướng (mỗi mục một file `.md`): `Strategy`, `Vision`, `Mission`, `Organization` (sơ đồ tổ chức), `Business_Model`, `KPI` (cấp Viện), `Digital_Strategy`, `Roadmap`, `Architecture` (kiến trúc tổng thể), `Glossary` (từ điển thuật ngữ doanh nghiệp)
- Sơ đồ tổ chức, mô hình kinh doanh, bản đồ chiến lược (.md, .pptx, hình ảnh)

**KHÔNG lưu ở đây:**
- Quy trình tác nghiệp → `04_PROCESS_LIBRARY`
- Biểu mẫu → `06_SHARED_RESOURCES/Forms`
- Hồ sơ khách hàng, kết quả đo → ManLab/cơ sở dữ liệu vận hành

**Lưu ý:** Tài liệu ở đây ít thay đổi, mang tính định hướng; cập nhật khi xem xét của lãnh đạo.
