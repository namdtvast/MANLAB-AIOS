## Quy ước lưu trữ

- **Định dạng:** PNG, vector SVG hoặc tương đương
- **Độ phân giải:** ≥ 1920×1080 (phù hợp trình chiếu, in A3)
- **Kích thước tệp:** < 2MB/file (tối ưu truyền tải)
- **Gitignore:** Các file PNG không được push lên git; lưu local hoặc trên ổ chia sẻ
