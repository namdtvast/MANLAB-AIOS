# 12 — RESEARCH (Nghiên cứu & Phát triển)

**Ý nghĩa:** Không gian R&D, thử nghiệm ý tưởng — tách khỏi hệ thống vận hành chính thức.

| Thư mục con | Lưu gì |
|---|---|
| `KC4.0` | Đề tài/chương trình KH&CN 4.0. |
| `DMC` | Chứng chỉ đo lường số: DCC (hiệu chuẩn), DVC (kiểm định), DTC (thử nghiệm). |
| `OCR` | Nghiên cứu nhận dạng ký tự/tài liệu. |
| `AI` | Thử nghiệm mô hình/ứng dụng AI. |
| `GPS` | Định vị, dữ liệu hiện trường. |
| `Paper` | Bài báo khoa học. |
| `Patent` | Sáng chế, giải pháp hữu ích. |
| `Proposal` | Thuyết minh đề tài, đề xuất. |
| `Pilot` | Triển khai thí điểm. |

**Lưu ý:** Khi một nghiên cứu trưởng thành → chuyển hóa thành Capability/MP/M ở các tầng vận hành.
