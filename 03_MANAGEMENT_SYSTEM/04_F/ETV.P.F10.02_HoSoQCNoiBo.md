## Kết quả & đánh giá

| Chỉ số | Công thức | Giá trị đo | Ngưỡng/tiêu chí | Kết luận |
|---|---|---|---|---|
| z-score | z = (x − xpt)/σpt | | \|z\| ≤ 2 | ☐ Đạt ☐ Cảnh báo ☐ Không đạt |
| En | En = (x − Xref)/√(U²lab+U²ref) | | \|En\| ≤ 1 | ☐ Đạt ☐ Không đạt |
| Bias % | 100×(x − xref)/xref | | theo PP | |
| Recovery % | 100×(Cs − C0)/Cadd | | theo PP/nền mẫu | |
| RSD/CV % | 100×SD/Mean | | theo PP | |
| Biểu đồ kiểm soát | đường TT + giới hạn cảnh báo/hành động | | rule đã phê duyệt | |

> Tiêu chí của phương pháp/tiêu chuẩn/cơ quan công nhận/khách hàng được **ưu tiên**. Giá trị mặc định chỉ dùng khi QLCL/LĐP phê duyệt trước.
