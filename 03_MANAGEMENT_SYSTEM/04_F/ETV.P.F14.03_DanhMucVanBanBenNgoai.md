# ETV.P.F 14.03 — Danh mục văn bản bên ngoài

> Biểu mẫu gốc — bản trắng. Thuộc `ETV.P14` §6.1, §10.2. Danh sách sống thực tế quản lý trên ManLab.

| Mã số nội bộ (`id`) | Số hiệu văn bản gốc | Tên văn bản | Loại | Cơ quan ban hành | Ngày ban hành | Ngày tiếp nhận | Người xử lý | Trạng thái áp dụng (`status`) | Phạm vi áp dụng tại ETV | Thay thế văn bản nào | Ghi chú |
|---|---|---|---|---|---|---|---|---|---|---|---|
| | | | | | | | | | | | |

Loại văn bản bên ngoài (giá trị hợp lệ): VBQPPL (Luật/Nghị định/Thông tư/Quyết định/Công văn) · QCVN/TCVN/ĐLVN · Văn bản của cơ quan công nhận/chỉ định · Quyết định cấp cho ETV · Chứng chỉ đào tạo · Khác.

> Khi văn bản bên ngoài được thay thế (hết hiệu lực theo văn bản mới), cập nhật cột "Trạng thái áp dụng" → `Hết hiệu lực` và rà soát các văn bản nội bộ có `legal_basis` liên quan theo `ETV.P14` §15.
