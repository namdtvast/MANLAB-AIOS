# 03 — MANAGEMENT SYSTEM (Chuẩn mực)

**Ý nghĩa:** Nơi đặt 'luật chơi' — chuẩn mực ISO và Sổ tay chất lượng. Mô tả Viện phải tuân thủ cái gì, KHÔNG mô tả làm thế nào.

**Lưu file gì ở đây:**
- Mỗi tiêu chuẩn một thư mục: `ISO9001`, `ISO17025`, `ISO17034`, `ISO27001`, `ISO42001` — chứa bản kiểm soát/đối chiếu điều khoản, ghi chú áp dụng, ánh xạ điều khoản
- `ETV.QM` — Sổ tay chất lượng (tài liệu cấp cao nhất)

**KHÔNG lưu ở đây:**
- **Quy trình MP** (đó là tầng `04`)
- Biểu mẫu, hồ sơ (→ `06`, `11`)
- Bản scan tiêu chuẩn vi phạm bản quyền — chỉ lưu ghi chú/đối chiếu, không lưu bản full nếu license không cho phép

**Lưu ý:** Phân biệt rạch ròi: tầng này = 'chuẩn mực cần đạt'; tầng 04 = 'cách thực hiện'. Không trộn lẫn.
