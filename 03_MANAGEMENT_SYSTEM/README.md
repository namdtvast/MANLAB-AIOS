# 03 — MANAGEMENT SYSTEM (Chuẩn mực)

**Ý nghĩa:** Nơi đặt 'luật chơi' — chuẩn mực ISO và Sổ tay chất lượng. Mô tả phải tuân thủ gì, KHÔNG mô tả làm thế nào.

**Lưu file gì ở đây:**
- 01_ETV.QM (Sổ tay)
- 02_ISO9001 … 06_ISO42001 — bản đối chiếu điều khoản, ghi chú áp dụng

**KHÔNG lưu ở đây:**
- **Quy trình MP** (→04)
- Biểu mẫu/hồ sơ (→06/11)
- Bản scan tiêu chuẩn vi phạm bản quyền

**Lưu ý:** Phân biệt: tầng này = 'chuẩn mực cần đạt'; tầng 04 = 'cách thực hiện'.
