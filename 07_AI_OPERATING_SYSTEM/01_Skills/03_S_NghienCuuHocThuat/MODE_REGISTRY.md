# Mode Registry

| Mã | Chế độ | Mục đích |
|---|---|---|
| guided | Hỏi từng bước | Người dùng chưa rõ yêu cầu |
| direct | Thực hiện trực tiếp | Đầu vào đã đầy đủ |
| deep-research | Nghiên cứu sâu | Tổng quan và evidence map |
| systematic-review | PRISMA | Tổng quan hệ thống |
| manuscript | Viết bài | Bài báo quốc tế/quốc gia |
| proposal | Đề xuất/thuyết minh | Hồ sơ đề tài các cấp |
| report | Báo cáo | Chuyên đề, tiến độ, tổng kết |
| reviewer | Phản biện chỉ đọc | Đánh giá độc lập |
| revision | Sửa có truy vết | Response-to-reviewers |
| integrity-audit | Kiểm toán | Claim, citation, data, provenance |
| reproducibility | Đóng gói tái lập | Dữ liệu, code, cấu hình, phiên bản |

Mỗi phiên phải ghi mode hiện hành trong `PROJECT_PROFILE.md`.
