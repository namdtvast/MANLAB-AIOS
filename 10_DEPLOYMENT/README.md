# 10 — DEPLOYMENT (Triển khai & vận hành)

**Ý nghĩa:** Cấu hình đưa hệ thống lên môi trường chạy thật và duy trì vận hành.

| Thư mục con | Lưu gì |
|---|---|
| `Docker` | Dockerfile, compose, image build. |
| `K8S` | Manifest Kubernetes, Helm chart. |
| `Cloud` | Cấu hình hạ tầng đám mây. |
| `Monitoring` | Giám sát, cảnh báo, log, metrics. |
| `Backup` | Chính sách và script sao lưu. |
| `Disaster_Recovery` | Kế hoạch và quy trình khôi phục thảm họa. |

**Lưu ý:** Toàn bộ nên ở dạng cấu hình-như-mã (IaC), có kiểm soát phiên bản.
