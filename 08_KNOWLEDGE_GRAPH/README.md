# 08 — KNOWLEDGE GRAPH ⭐

**Ý nghĩa:** Không phải 'Knowledge' đơn thuần — mà là tri thức có cấu trúc để **AI đọc**: quy định, tiêu chuẩn, ontology, vector.

| Thư mục con | Lưu gì |
|---|---|
| `Regulations` | Văn bản quy phạm pháp luật: luật, nghị định, thông tư. |
| `ISO` | Tri thức/giải thích về các tiêu chuẩn ISO. |
| `DLVN` | Văn bản kỹ thuật đo lường Việt Nam (ĐLVN). |
| `TCVN` | Tiêu chuẩn quốc gia liên quan. |
| `ILAC` | Tài liệu ILAC/APAC và công nhận quốc tế. |
| `FAQ` | Câu hỏi thường gặp đã chuẩn hóa. |
| `Case_Study` | Tình huống thực tế, ví dụ áp dụng. |
| `Ontology` | Bản thể học: khái niệm và quan hệ giữa chúng. |
| `Embedding` | Vector nhúng của tài liệu/tri thức. |
| `Vector_DB` | Cơ sở dữ liệu vector phục vụ tra cứu ngữ nghĩa. |
| `Lessons_Learned` | Bài học kinh nghiệm, sai sót đã gặp. |

**Lưu ý:** Nội dung nên ở dạng văn bản/markdown để dễ index cho AI. Phân biệt: tầng 03 = 'chuẩn mực kiểm soát', tầng 08 = 'tri thức để AI tham khảo'.
