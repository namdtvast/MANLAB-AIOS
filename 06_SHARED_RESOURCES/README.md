# 06 — SHARED RESOURCES ⭐ (Tài nguyên dùng chung)

**Ý nghĩa:** Tài nguyên dùng chung cho mọi MP/M — nguyên tắc 'một nguồn sự thật'. Không thuộc riêng quy trình hay module nào.

| Thư mục con | Lưu gì |
|---|---|
| `Forms` | Biểu mẫu gốc (F-xxx) dùng chung — bản trắng, chưa điền. |
| `Templates` | Mẫu tài liệu: báo cáo, công văn, slide, trình bày. |
| `Reference_Data` | Dữ liệu tham chiếu: bảng tra, danh mục đại lượng, hằng số. |
| `Master_Data` | Dữ liệu chủ: danh mục mã hóa dùng chung toàn hệ thống. |
| `Units` | Đơn vị đo theo SI và bảng quy đổi. |
| `Customers` | Danh mục khách hàng (master) — KHÔNG phải hồ sơ giao dịch. |
| `Equipment` | Danh mục thiết bị, chuẩn đo lường (master). |
| `Personnel` | Danh mục nhân sự và ma trận năng lực (master). |
| `Methods` | Danh mục phương pháp/quy trình kỹ thuật được áp dụng. |
| `Standards` | Chỉ mục tiêu chuẩn/ĐLVN viện dẫn — không phải bản full (xem 03/08). |
| `OCR_Models` | Mô hình OCR, template nhận dạng tài liệu/thiết bị. |
| `Icons` | Bộ biểu tượng dùng chung cho giao diện. |
| `Branding` | Logo, màu sắc, phông chữ, bộ nhận diện thương hiệu. |

**KHÔNG lưu trong cả tầng này:**
- Hồ sơ đã điền hoặc dữ liệu giao dịch thật
- Bản nội dung tiêu chuẩn đầy đủ (ở 03/08)

**Lưu ý:** Sửa một chỗ ở đây → mọi MP/M dùng lại đều cập nhật. Đây là lý do tránh nhân bản biểu mẫu.
