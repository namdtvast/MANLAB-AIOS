# 06 — SHARED RESOURCES ⭐ (Tài nguyên dùng chung)

**Ý nghĩa:** Tài nguyên dùng chung cho mọi MP/M — 'một nguồn sự thật'.

| Thư mục con | Lưu gì |
|---|---|
| `01_Forms` | Biểu mẫu gốc (F-xxx), bản trắng chưa điền |
| `02_Templates` | Mẫu báo cáo, công văn, slide |
| `03_Reference_Data` | Bảng tra, danh mục đại lượng, hằng số |
| `04_Master_Data` | Dữ liệu chủ mã hóa dùng chung |
| `05_Units` | Đơn vị SI và bảng quy đổi |
| `06_Customers` | Danh mục khách hàng (master), không phải hồ sơ giao dịch |
| `07_Equipment` | Danh mục thiết bị, chuẩn (master) |
| `08_Personnel` | Danh mục nhân sự & ma trận năng lực |
| `09_Methods` | Danh mục phương pháp/quy trình kỹ thuật |
| `10_Standards` | Chỉ mục tiêu chuẩn/ĐLVN viện dẫn |
| `11_OCR_Models` | Mô hình OCR, template nhận dạng |
| `12_Icons` | Bộ biểu tượng giao diện |
| `13_Branding` | Logo, màu, font, nhận diện |

**KHÔNG lưu trong tầng này:**
- Hồ sơ đã điền/đã phát hành (→11)
- Dữ liệu giao dịch thật (→CSDL/ManLab)

**Lưu ý:** Sửa một chỗ → mọi nơi dùng lại cập nhật. Đây là lý do tránh nhân bản.
