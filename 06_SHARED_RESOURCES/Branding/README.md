# Branding

**Ý nghĩa:** Logo, màu sắc, phông chữ, bộ nhận diện thương hiệu.

**Lưu file gì ở đây:**
- Nội dung thuộc loại: logo, màu sắc, phông chữ, bộ nhận diện thương hiệu.

**KHÔNG lưu ở đây:**
- Hồ sơ đã điền/đã phát hành (→ hồ sơ vận hành, `11_COMPLIANCE`)
- Dữ liệu giao dịch thật (→ CSDL/ManLab)

**Lưu ý:** Sửa một chỗ ở đây → mọi MP/M dùng lại đều cập nhật. Đây là lý do tránh nhân bản biểu mẫu.
